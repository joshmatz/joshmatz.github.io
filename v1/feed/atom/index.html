]]> 0 Josh Matz http://www.joshmatz.com <![CDATA[Quora: The Q&A website that does it right]]> http://www.joshmatz.com/?p=184 2011-01-24T05:23:40Z 2011-01-20T05:19:14Z I decided to try out Quora for the first time this past week and, ya know, it’s pretty good! I’ve used other websites (Stack Overflow, Yahoo Answers) but have never found myself going back to them just to see more questions and answers.

I started my account by hooking in Facebook first and after allowing it access I was then prompted for my email address and a password so I could login next time.

Once they had my email and password I was shown a page that looked very much like this:


These are the results my girlfriend got when we signed her up. Pretty much everything listed above was spot on for her, expect maybe Rock Music and Play. Because I’m in Graphic Design student, Quora gave me very relevant topics such as the obvious, HTML, CSS and Graphic Design. If you click the “Follow” link on any of the present options, a new topic replaces the item you followed. I clicked “Follow” a good 10 times because the suggestions were all so good!


The interface is super intuitive. I’m not sure the particular algorithm that Quora uses to decide which questions have an answer displayed with them, but it works very well. It displays an abbreviated portion of the answer and if you’d like to read more, just click “(more)” and the full answer appears without loading a separate page.

Quora definitely has me enthralled. The answers posted have been insightful from the lot I’ve seen and the interface shines in all it’s AJAXy gloriousness.

If you’re curious about things or find yourself needing a question answered, try Quora!
